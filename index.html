Projects

  • Medical Visual Question Answering

    This project involves developing a medical visual question answering (VQA) system using the Path VQA dataset, which includes histopathology images and related questions. Utilizing the ViLT (Vision and Language Transformer) model, the system takes pathology or radiology images and answers questions related to it.

  • Web-Scraping-and-PDF-Data-Extraction

    This Python script performs PDF and web data scraping, transforms the extracted data, stores it in a database, and provides API endpoints for data storage and downloading.

  • Urdu Text Sentiment Analysis

    This project focuses on sentiment analysis of Urdu language text data using deep learning techniques. The dataset used in this project contains Urdu movie reviews collected from IMDb. The goal is to classify the sentiment of the reviews as positive or negative.
